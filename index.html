⚠️ Firebase ڕێکنەخراوە

تکایە ئەم هەنگاوانە بەجێبهێنە بۆ چالاک کردنی یاری ئۆنلاین

1 بچوو بۆ Firebase Console
console.firebase.google.com/project/spygames-c2744
2 Realtime Database دروست بکە
لە لیستی چەپ کرتە بکە: Build → Realtime Database
دوای ئەوە کرتەی "Create Database" بکە
ناوچەی: United States هەڵبژێرە
Mode: Test Mode هەڵبژێرە
3 قانوونەکانی ئازادانە دابنێ
لە ناو Realtime Database کرتەی Rules بکە، ئەم کۆدەی خوارەوەی تێدا بنووسە:
{ "rules": { ".read": true, ".write": true } }
دوای ئەوە Publish بکە
4 databaseURL زیاد بکە
لە ناو Project Settings دابمەزرێنە، URL ی database copy بکە وەک:
https://spygames-c2744-default-rtdb.firebaseio.com
